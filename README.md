# vk posts parser
This console application analyzes data from VKontakte groups and public pages.

## Installation
```sh
$ git clone https://github.com/vsezol/vk-posts-parser
$ yarn / npm i
$ node index
```

## How to use?
```sh
$ Set the url: https://vk.com/example_group
$ Set the file name: example_data.txt
```
